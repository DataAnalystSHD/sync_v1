SHD Sync คืออะไร

เครื่องมือคัดลอกข้อมูลจาก ต้นทาง ไป ปลายทาง ให้ตรงกันอัตโนมัติ — เชื่อมได้ทั้ง Google Sheet, Lark Base และ Lark Sheet ทั้งซิงค์เองครั้งเดียว หรือตั้งเวลาให้ระบบทำให้ตลอด 24 ชม.

เริ่มต้นใช้งาน

  1. กดปุ่ม Login มุมขวาบน แล้วเข้าสู่ระบบด้วยบัญชี Google ที่มีสิทธิ์เข้าถึงชีต
  2. เมื่อขึ้นอีเมลสีเขียวมุมขวา = พร้อมใช้งาน

ซิงค์ข้อมูล (Sync Now)

  1. วาง ลิงก์ต้นทาง (Google Sheet URL) และ ลิงก์ปลายทาง (Lark / Feishu Base URL)
  2. เลือก Sync Direction — ทิศทางว่าจะเอาข้อมูลจากไหนไปไหน
  3. Columns (ไม่บังคับ) — กด “สแกน & เลือกคอลัมน์” เพื่อเลือกเฉพาะคอลัมน์ที่อยากซิงค์
  4. Filter (ไม่บังคับ · เฉพาะต้นทางเป็น Lark Base) — กรองเอาเฉพาะบางแถวตามค่า dropdown
  5. เลือก Sync Mode — Replace หรือ Append
  6. กด Sync Now รอจนขึ้นผลลัพธ์ว่าสำเร็จ

ทิศทางการซิงค์ (Direction)

Google Sheet → Lark Base ชีต → ตาราง Lark · ลิงก์ในข้อความคลิกได้
Lark Base → Google Sheet ตาราง Lark → ชีต
Lark Sheet → Lark Base Lark Sheet → ตาราง Lark
Lark Base → Lark Sheet ตาราง Lark → Lark Sheet
Google Sheet → Lark Sheet ชีต → Lark Sheet · คงลิงก์ในเซลล์
Lark Sheet → Google Sheet Lark Sheet → ชีต

โหมดซิงค์ (Sync Mode)

Replace ลบข้อมูลปลายทางทิ้งทั้งหมด แล้วซิงค์ใหม่ ใช้เมื่ออยากให้ปลายทางเหมือนต้นทางเป๊ะ ทุกครั้ง · จำเป็นถ้าต้องการให้คอลัมน์ที่มีลิงก์กลายเป็นลิงก์คลิกได้
Append เพิ่มต่อท้ายข้อมูลเดิม (ไม่ลบ) ใช้เมื่ออยากสะสมข้อมูลใหม่ต่อจากของเดิม โดยไม่แตะแถวที่มีอยู่แล้ว

ตัวเลือกเสริม

เลือกคอลัมน์ สแกนต้นทางแล้วติ๊กเฉพาะคอลัมน์ที่ต้องการ — ไม่เลือก = ซิงค์ทุกคอลัมน์
ตัวกรอง (Filter) เลือกเอาเฉพาะแถวที่ค่าตรงกับ dropdown ที่กำหนด (เฉพาะต้นทาง Lark Base)
แถวแรกไม่มีหัวคอลัมน์ ติ๊กเมื่อแถวแรกของชีตไม่ใช่หัวตาราง — ระบบจะตั้งชื่อ Column 1, 2, 3… ให้เอง (ใช้เมื่อปลายทางเป็น Sheet)

ตั้งซิงค์อัตโนมัติ (Auto-sync)

  1. กรอกลิงก์ + ตั้งค่าเหมือนการซิงค์ปกติ
  2. เลือกช่วงเวลาในช่อง Auto-sync ทุก ๆ (เช่น ทุก 3 นาที / ทุก 1 ชม. / ทุก 1 วัน หรือพิมพ์เองก็ได้)
  3. กด Save Auto-sync — ระบบจะซิงค์ให้เองตามรอบ ตลอด 24 ชม. โดยไม่ต้องเปิดเว็บค้าง
  4. ดู/แก้/ลบงานที่ตั้งไว้ได้ที่แท็บ Auto-sync

ประวัติการซิงค์ (History)

แท็บ History แสดงว่าซิงค์อะไร เมื่อไหร่ กี่แถว สำเร็จหรือไม่ — คลิกแต่ละรายการเพื่อดูรายละเอียด และลบประวัติได้

ข้อควรรู้

อยากให้ ลิงก์ที่ฝังในข้อความ (เช่นชื่อ KOL) กลายเป็นลิงก์คลิกได้ใน Lark Base → ต้องใช้โหมด Replace · ระบบจะแปลงคอลัมน์ที่เซลล์ส่วนใหญ่มีลิงก์เป็น URL field ให้เอง
Lark URL field เก็บ “ข้อความล้วน” ไม่ได้ — คอลัมน์ที่มีลิงก์ปน ๆ กันจะคงเป็นข้อความ เพื่อไม่ให้ชื่ออื่นกลายเป็นลิงก์ปลอม
