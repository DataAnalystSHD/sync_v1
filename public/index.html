ตั้งค่า Lark App

1
Add app เข้า Bitable / Sheet (ครั้งเดียวต่อไฟล์)
เปิด Bitable / Sheet ที่ต้องการ → กดเมนู ⋯ (3 จุด) มุมขวาบน → More → Add Applications → พิมพ์ค้น sync hub → กด Add
ไม่เห็นเมนู Add Applications? สิทธิ์ไม่พอ
เมนูนี้เห็นเฉพาะคนที่มีสิทธิ์ Can manage หรือเจ้าของไฟล์ — ถ้ามีแค่สิทธิ์ Edit ให้ส่งลิงก์ไฟล์ให้เจ้าของกด Add ให้ครั้งเดียว จากนั้นทุกคนตั้ง sync กับไฟล์นั้นได้เลย ไม่ต้องขอสิทธิ์เพิ่ม
ทีมใช้กันหลายไฟล์? แนะนำ
ให้ admin เพิ่มแอปที่ระดับโฟลเดอร์ / Shared Space ครั้งเดียว — ไฟล์ทั้งหมดข้างใน (รวมไฟล์ที่สร้างใหม่ในอนาคต) ใช้ sync ได้อัตโนมัติ ไม่ต้องไล่เพิ่มทีละไฟล์
